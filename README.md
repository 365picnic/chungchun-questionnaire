# 청춘한의원 사전문진표

환자가 태블릿이나 본인 휴대폰으로 접속해 사전 문진표를 작성하면, 답변이 구글 시트에 자동으로 쌓이는 방식입니다. 원장님은 구글 시트를 열어서 확인하시면 됩니다.

## 구조

- `docs/index.html` — 환자가 작성하는 문진표 화면. GitHub Pages로 배포됩니다.
- `google-apps-script/Code.gs` — 문진 답변을 구글 시트에 저장해주는 스크립트.
- `app.py`, `static/`, `questionnaire.db` — 이전에 만들었던 서버(FastAPI+SQLite) 버전. 지금은 사용하지 않지만 참고용으로 남겨두었습니다. `questionnaire.db`는 환자 정보가 들어있어 깃허브에는 올리지 않습니다.

## 처음 설정하는 방법 (원장님이 직접 하실 부분)

### 1. 구글 시트 만들고 Apps Script 연결하기

1. 구글 드라이브에서 새 스프레드시트를 만듭니다. (예: "청춘한의원 문진 응답")
2. 상단 메뉴에서 **확장 프로그램 > Apps Script** 를 클릭합니다.
3. 열린 편집기에 원래 있던 코드를 지우고, 이 저장소의 `google-apps-script/Code.gs` 내용을 그대로 복사해 붙여넣습니다.
4. 저장(Ctrl+S) 합니다.
5. 우측 상단 **배포 > 새 배포** 클릭.
6. 유형에서 **웹 앱** 선택.
   - 실행할 사용자: **나(본인 구글 계정)**
   - 액세스 권한이 있는 사용자: **모든 사용자**
7. **배포** 클릭 → 권한 승인 화면이 뜨면 본인 계정으로 로그인하고 허용합니다.
8. 배포가 끝나면 나오는 **웹 앱 URL**을 복사해둡니다. (`https://script.google.com/macros/s/.../exec` 형태)

### 2. 문진표 페이지에 구글시트 주소 연결하기

`docs/index.html` 파일에서 아래 줄을 찾아, 위에서 복사한 웹 앱 URL로 바꿔주세요.

```js
const GAS_URL = "REPLACE_WITH_GOOGLE_APPS_SCRIPT_URL";
```

### 3. GitHub Pages로 공개 주소 만들기

저장소 설정(Settings) > Pages 메뉴에서 소스 브랜치를 `main`, 폴더를 `/docs`로 지정하면 몇 분 안에 아래와 같은 주소가 생성됩니다.

```
https://365picnic.github.io/chungchun-questionnaire/
```

이 주소를 환자에게 QR코드나 링크로 안내하면 됩니다.

### 4. 답변 확인하기

원장실 컴퓨터에서 1번에서 만든 구글 시트를 열면, 환자가 제출할 때마다 새로운 줄이 자동으로 추가됩니다. "문진 요약" 칸에 읽기 쉬운 형태로 정리되어 있습니다.

## 참고

- 문진표 코드(질문 목록, 디자인)는 공개 저장소에 있지만, 환자가 실제로 입력한 답변은 저장소에 저장되지 않고 구글 시트에만 저장됩니다.
- 문항 내용은 `docs/index.html` 안의 `Q` 배열에서 수정할 수 있습니다.
